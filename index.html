Features

Plugin system

Everything is a plugin so everything can be changed by just installing a new plugin.

Distributed

Designed for sharing data between machines. No need to run it all on a single server.

Versatile

Can take data from many sources and stream it on-demand, no need to complete download or extract files.

No tracking

Nobody knows what you're doing unless you tell them. No tracking, no "telemetry" or any other garbage.
